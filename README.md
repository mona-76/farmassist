## 🚀 HOW TO RUN

### **Simple Method (Recommended)**
1. Double-click `LAUNCH_APP.bat`
2. Wait for the browser to open automatically
3. Start using the application!

### **Manual Method**
1. Open terminal in project folder
2. Start backend: `cd backend && node index.js`
3. Start frontend: `cd frontend && npm run dev`
4. Open browser: `http://localhost:3005`

## 🔧 TROUBLESHOOTING

### **Port Already in Use**
- Run `LAUNCH_APP.bat` - it automatically cleans up old processes

### **Backend Not Connecting**
- Ensure backend is running on port 5000
- Check terminal for error messages

### **Translation Not Working**
- Clear browser cache
- Refresh the page
- Check language selector

### **Voice Input Not Working**
- Use Chrome or Edge browser
- Allow microphone permissions
- Check system microphone settings
